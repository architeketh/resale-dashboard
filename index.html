Resale command center

ThreadUp and Depop inventory in one place.

This dashboard reads public JSON from architeketh/resale-dashboard. The updated scrapers write directly into the repo, and the weekly capture tool lets you save a Sunday through Saturday sales snapshot for each platform.

How it works now
›

1. Save the token once

The first time either scraper or weekly capture runs it uses your GitHub token with repo write access. The token stays in your browser on your device.

2. Install the scraper buttons

Drag these links to your bookmarks bar once, then click the matching bookmark while you are on ThreadUp or Depop.

Install ThreadUp Scraper

Install Depop Scraper

3. Capture weekly sales

Use the capture button in each platform header to save that platform's sold count and revenue for the current Sunday through Saturday week.

  1. ThreadUp writes data/threadup.json
  2. Depop writes data/depop.json
  3. Weekly snapshots write data/weekly-sales.json
