Workflow

How it works now

1. Save the token once

The first time either scraper runs it asks for a GitHub personal access token with repo write access. The token is stored only in your browser on your device.

2. Install the scraper buttons

Drag each button below to your bookmarks bar once. After that, open ThreadUp or Depop and click the matching bookmark to run the scraper.

Install ThreadUp Scraper

Install Depop Scraper

3. Run and refresh

On ThreadUp or Depop, click the matching bookmarklet. It writes directly to GitHub, then you come back here and refresh.

  1. ThreadUp writes data/threadup.json
  2. Depop writes data/depop.json
  3. Both update data/sync-status.json
