How it works now
+

1. Install the scraper buttons

Drag these links to your bookmarks bar once, then click the matching bookmark while you are on ThreadUp or depop.

Install ThreadUp Scraper

Install depop Scraper

2. Sync inventory

Each scraper sends inventory to your Supabase resale-sync Edge Function, which updates the resale_inventory table.

3. Capture weekly sales

Use the platform capture buttons after your scrapes are current to save weekly sold count and revenue in resale_weekly_snapshots.
